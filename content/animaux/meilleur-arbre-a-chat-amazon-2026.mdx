L'arbre a chat est indispensable pour les chats d'interieur. Il leur offre un espace pour grimper, griffer, se cacher et observer. Voici les meilleurs sur Amazon.

## Comment choisir son arbre a chat

- **Taille** : petit (60-80 cm) pour un chaton, moyen (100-140 cm) pour un chat, grand (160 cm+) pour les grimpeurs
- **Stabilite** : une base large et lourde evite le basculement
- **Materiaux** : sisal pour les griffoirs (le plus resistant), peluche pour le confort
- **Nombre de chats** : plus de chats = plus de plateformes et de niches
- **Espace** : mesurez l'espace disponible avant d'acheter
