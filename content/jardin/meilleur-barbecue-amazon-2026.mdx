Le barbecue est l'incontournable de la belle saison. Charbon pour les puristes, gaz pour le pratique, electrique pour le balcon : chaque type a ses avantages. Voici les meilleurs modeles sur Amazon pour tous les budgets.

## Comment choisir son barbecue

- **Type** : charbon (gout fume authentique), gaz (allumage rapide, temperature precise), electrique (balcon/appartement)
- **Surface de cuisson** : 40-50 cm pour 4 personnes, 55 cm+ pour 6-8 personnes
- **Materiau** : acier emaille (budget), inox (durable), fonte (retention chaleur)
- **Couvercle** : indispensable pour les cuissons longues et le roti
- **Mobilite** : roulettes, compacite, poids — selon votre espace
